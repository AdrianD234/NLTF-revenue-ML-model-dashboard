# QUALITY_RUBRIC.md

Feature-completeness quality scores after the strict audit and repair pass. Minimum acceptable score is 4/5.

| Category | Score | Evidence |
|---|---:|---|
| Data loading robustness | 5/5 | Alias, workbook, empty-file, stream-label, and live-run exclusion tests. |
| Executive summary usefulness | 5/5 | KPI cards, finalist MAPE chart, and management answer cards for winner, Schiff result, robustness, and warnings. |
| Candidate landscape usefulness | 5/5 | Interactive qMAPE/aMAPE scatter with stream colours, finalist and Schiff markers, frontier-focused axis, and top-candidate table. |
| Schiff comparison completeness | 5/5 | Decision cards, paired table, gain chart, scatter, best challenger summary, and structural benchmark chart. |
| Ensemble composition clarity | 5/5 | Weight/membership chart, C-label mapping, finalist filtering, method readout, and optional origin path where data exists. |
| Forecast/error explorer usefulness | 5/5 | Compact controls, actual-versus-predicted chart, percent-error chart, horizon-bucket distribution, and horizon MAPE drilldown. |
| Stress-test page usefulness | 5/5 | Required horizon/stress buckets and Light RUC interpretation. |
| Model inventory usefulness | 5/5 | Filtering, ranking, inventory KPI cards, best-row readout, full candidate table, supporting tables, and CSV download. |
| Run audit usefulness | 5/5 | File status, feature counts, feature audit, logged errors, and explicit run-health flags. |
| Visual polish | 5/5 | Light governance-report theme, compact tabs, shortened long labels, chart-first pages, cards, collapsed detail rows, and screenshot-reviewed pages. |

All categories meet or exceed 4/5.
