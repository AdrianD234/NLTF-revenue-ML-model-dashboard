# Dashboard Quality Rubric

Validation standard: product-hardening sprint threshold is at least 9/10 per page in `artifacts/deep_quality_review.md`; each legacy 5-point category must score at least 4/5.

| Category | Score | Evidence |
|---|---:|---|
| Data loading robustness | 5/5 | Alias, workbook, empty-file, stream-label, live-run exclusion, and percent-unit diagnostics. |
| Executive summary usefulness | 5/5 | KPI cards, management conclusion, decision status, data-quality warning panel, finalist MAPE chart, export. |
| Candidate landscape usefulness | 5/5 | Frontier trace, outlier toggle, finalists, Schiff markers, hover detail, and annotations. |
| Schiff comparison completeness | 5/5 | Decision cards, pure-Schiff classifier, gain chart, paired scatter, and benchmark chart. |
| Ensemble composition clarity | 5/5 | Method readout, C-label mapping, component chart, origin path where available, static/prequential caveat. |
| Forecast/error explorer usefulness | 5/5 | Actual/predicted, percent error, horizon-bucket distribution, and horizon MAPE drilldown. |
| Stress-test page usefulness | 5/5 | Required buckets plus 2020-21, high-risk guide, risk band, and RUC interpretation. |
| Model inventory usefulness | 5/5 | KPI cards, ranking options, model detail, family performance chart, Schiff-class mix, CSV download. |
| Run audit usefulness | 5/5 | Run-health cards, file status, feature chart, error-type chart, and diagnostics. |
| Visual polish | 5/5 | Report theme, final screenshots, collapsed operational chrome, chart-first pages, shortened labels. |
